KEY FEATURES BY FILE
====================

📄 types/index.ts
   - Card, CharacteristicRange, Position
   - PlacedCard, CalculatedValues
   - AppState

📄 utils/calculations.ts
   - getCharacteristicValue()
   - calculateRangeValue()
   - calculateCardValues()
   - getCharacteristicLabel()

📄 components/CardForm.tsx
   - Create/edit cards
   - Add/remove characteristic ranges
   - Form validation

📄 components/CardList.tsx
   - Display all cards
   - Delete cards
   - Show card details

📄 components/AtlasGrid.tsx
   - 8×8 grid rendering
   - Drag & drop handling
   - Card placement logic
   - Visual feedback

📄 components/CardSelector.tsx
   - Display available cards
   - Drag initiation
   - Card preview

📄 components/PlacedCardInfo.tsx
   - Show placed cards
   - Display calculated values
   - Position information

📄 pages/Collection.tsx
   - Card management interface
   - Two-column layout
   - Form + List integration

📄 pages/Atlas.tsx
   - Three-column layout
   - State management for placed cards
   - Clear atlas functionality

DATA FLOW
=========

1. User creates card in CardForm
   ↓
2. Card saved to App state (cards array)
   ↓
3. Card persisted to localStorage
   ↓
4. Card appears in CardList (Collection page)
   ↓
5. Card available in CardSelector (Atlas page)
   ↓
6. User drags card to AtlasGrid
   ↓
7. calculations.ts calculates values based on position
   ↓
8. PlacedCard created with calculated values
   ↓
9. PlacedCard displayed in AtlasGrid
   ↓
10. Details shown in PlacedCardInfo
